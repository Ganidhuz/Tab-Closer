# Tab Closer

A sleek, modern Chrome extension that automatically closes tabs based on custom keywords in their title or URL. Built with Tailwind CSS and designed with a Shadcn-inspired aesthetic.

## Features

- **Keyword Matching**: Automatically closes tabs containing specified keywords.
- **Dark/Light Mode**: Full theme support with persistence.
- **Modern UI**: Clean, responsive interface built with Tailwind.
- **Popup & Options**: Access settings directly from the extension icon or in a dedicated options page.
- **Sync**: Keywords and theme preferences are synced across your Chrome devices.

## Installation

1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" in the top right.
4. Click "Load unpacked" and select the `tab-closer-extension` folder.

## Usage

- Click the extension icon to open the quick-access popup.
- Add keywords (e.g., "youtube", "distract") to the list.
- Toggle between Light and Dark mode using the sun/moon icon in the top right.
